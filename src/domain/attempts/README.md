# Intentos medidos

Un intento medido conserva únicamente los datos necesarios para comparar progreso técnico:

- ejercicio y nota objetivo;
- fecha local y duración;
- trayectoria reducida de pitch;
- confianza y cobertura de voz;
- ataque, estabilización, error central, estabilidad y deriva final;
- calidad básica de captura;
- feedback priorizado;
- referencia al intento anterior.

El audio completo no se persiste por defecto.

Los reintentos solo se comparan cuando pertenecen al mismo ejercicio y a la misma frecuencia objetivo. Una mejora se calcula contra el intento anterior del propio usuario.
